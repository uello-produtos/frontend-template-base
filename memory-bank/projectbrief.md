# Project Brief

## Visão Geral

O **Frontend Template Base** é um template de inicialização moderno e completo para projetos frontend com Next.js. Foi criado para resolver o problema de inconsistência e retrabalho ao iniciar novos projetos, especialmente quando se trabalha com editores de código assistidos por IA como o Cursor.

## Objetivo Principal

Fornecer uma fundação confiável, padronizada e pronta para uso que acelere o início de qualquer aplicação frontend moderna, eliminando a necessidade de configurar do zero a cada novo projeto.

## Escopo do Projeto

### Incluído
- Stack completa configurada (Next.js 16, React 19, TypeScript)
- Sistema de UI com shadcn/ui e Tailwind CSS 4
- Sistema de animações com Framer Motion 12
- Gerenciamento de estado remoto com React Query
- Validação com Zod 4
- Formulários com react-hook-form
- Sistema de temas (claro/escuro)
- Tratamento de erros e loading states
- API client configurado
- Testes com Vitest
- Documentação para agentes de IA (AGENTS.md)

### Não Incluído (intencionalmente)
- Autenticação específica (deve ser implementada conforme necessidade)
- Banco de dados (template frontend apenas)
- Deploy específico (configuração genérica)
- Features de domínio específico

## Princípios de Design

1. **Produtividade**: Minimizar tempo de configuração inicial
2. **Consistência**: Padrões claros e documentados
3. **Manutenibilidade**: Código limpo e bem estruturado
4. **Escalabilidade**: Arquitetura que suporta crescimento
5. **Foco**: Energia direcionada para features, não setup

## Público-Alvo

- Desenvolvedores que iniciam projetos frontend frequentemente
- Equipes que precisam de consistência entre projetos
- Projetos que precisam de uma base sólida rapidamente
- Desenvolvedores que trabalham com IA assistida (Cursor, etc.)

## Requisitos Não-Funcionais

- TypeScript strict mode obrigatório
- Acessibilidade (WCAG) como padrão
- Performance otimizada (Server Components por padrão)
- Código limpo e documentado
- Testes configurados e exemplos fornecidos
